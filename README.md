```bash

# .env.local 파일에 ANTHROPIC_API_KEY 입력

# 개발 서버 시작
npm run dev
```

http://localhost:3000 에서 확인

## 사용 방법

### 1. 업종명 입력
생성할 인허가 업종명을 입력합니다.
- 예: `공장등록`, `근로자파견업`, `국제물류주선업`, `여행업`

### 2. 참고 자료 (선택)
해당 업종의 법률 조문, 등록 요건, 서류 목록 등을 붙여넣으면 정확도가 올라갑니다.
- 텍스트 직접 입력 또는 `.txt`, `.md` 파일 첨부 가능
- 참고 자료가 없어도 생성 가능하지만, 있으면 구체적 수치(자본금, 면적 등)가 정확해짐

### 3. 글 옵션 설정
| 옵션 | 설명 |
|------|------|
| 필수 포함 문구 | 글에 반드시 들어가야 할 키워드나 문구 |
| 추가 요청사항 | 특별히 강조할 내용이나 요청 |
| 본문 4 포함 | 체크 시 혜택/추가정보 섹션 추가 (5섹션 → 6���션) |
| LLM 선택 | Sonnet(추천, 고품질) / Haiku(빠름, 저비용) |

### 4. 생성 및 복사
- **글 생성하기** 클릭 → 1~2분 소요
- 결과 확인 후 3가지 복사 옵션:
  - **텍스트 복사**: 순수 텍스트
  - **리치텍스트 복사**: 네이버 블로그 에디터에 붙여넣기용 (서식 유지)
  - **HTML 복사**: 원본 HTML 코드

### 5. 네이버 블로그에 포스팅
1. "리치텍스트 복사" 클릭
2. 네이버 블로그 > 글쓰기 > 에디터에서 `Ctrl+V` 붙여넣기
3. 서식(제목, 굵기, 리스트)이 자동 적용됨

## 생성되는 글 구조

```
인트로          → 인사 + 업종 배경 + "이번 포스팅에서는..."
본문 1: ~이란   → 법적 정의 + 법률 인용 + 벌칙
본문 2: ~ 요건  → 자본금, 인력, 시설, 결격사유
본문 3: ~ 서류  → 필수/추가 서류 + 절차
(본문 4: 추가)  → 혜택, 인증, 변경등록 (선택)
엔딩            → 요약 + 전문가 의뢰 유도 CTA
```

## 기술 스택

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Claude API (Anthropic)

## 프로젝트 구조

```

features/llm/           → Claude API 추상화
features/blog/          → 글 생성 핵심 로직
  ├ style-profile.ts    → 인허가닷컴 스타일 프로파일 (26개 글 분석 기반)
  ├ prompt-builder.ts   → 프롬프트 조립
  └ blog-generator.ts   → 생성 오케스트레이터
components/             → UI 컴포넌트
docs/                   → 플랜 문서
```
